  • Propriedade de texto

    innerHTML

    Pega um texto de um elemento e reconhece o html que esta no elemento.

    li = "<li>Olá!</li>"
    const tarefa = li.

    innerHtml


    console.log("Tarefa é igual a:",tarefa)

    Tarefa é igual a: <li>Olá!</li>

    textContent

    Propriedade de nós, pega o texto de um elemento e o interpreta apenas como texto.

    li = "<li>Olá!</li>"
    const tarefa = li.

    textContent


    console.log("Tarefa é igual a:",tarefa)

    Tarefa é igual a: Olá!

    innerText

    Propriedade do html, pega o texto de um elemento e o interpreta apenas como texto.

    li = "<li>Olá!</li>"
    const tarefa = li.

    innerText


    console.log("Tarefa é igual a:",tarefa)

    Tarefa é igual a: Olá!

    Mouseenter

    Evento que acontesse quando o mouse se move sobre um elemento.

    const quadradosEnter = document.getElementsByClassName("quadrado")[0];
    quadradosEnter.addEventListener("mouseenter", () => {
    cor aleatoria const r = Math.floor(Math.random() * 256);

    });

    Mouseleave

    Evento que acontesse quando o mouse se move para fora de um elemento.

    const quadradosOut = document.getElementsByClassName("quadrado")[1];

    const g = Math.floor(Math.random() * 256);
    const b = Math.floor(Math.random() * 256);

    });
